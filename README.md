# AFP Lesion Paper

Analysis and figure-generation code for the AFP lesion manuscript examining lesion-associated changes in canary song structure, including phrase duration variability, syllable repetition, acoustic metrics, and latent-space similarity analyses.

## Repository structure

- `py_files/`: analysis and plotting scripts
- `figure_code/`: scripts used to generate manuscript figures
- `metadata/`: small metadata files required for analyses
- `figures/`: finalized or lightweight figure assets
- `docs/`: notes, run instructions, and method documentation

## Data availability

Large raw data files, audio files, `.npz` files, and generated outputs are not tracked in this repository. Paths in scripts may need to be updated for local use.

## Status

Private working repository for manuscript preparation.
